``` BASH
| 管道符
> 输出 大于号
< 输入 小于号

>> 追加输出
<< 追加输入

&>   正常错误等信息全部输出
```

管道符 == 万能门。将前一个命令的处理结果，当做参数交给后面的命令处理。

常见转义

``` BASH
\  反斜杠
'' 单引号
"" 双引号
`` 反引号 【把其中的命令执行后返回结果】
```

如果工作时，界面的文字出现乱码

``` BASH
LANG  系统语言、语系名称
```

别名 `alias` 删除

``` BASH
unset  别名
```
